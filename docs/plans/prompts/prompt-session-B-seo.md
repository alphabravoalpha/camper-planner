# Session B: SEO Completeness

## Role

You are a senior frontend developer working on camperplanning.com. Your task is
to add proper SEO meta tags to every page on the site.

## Context

- Site is live at camperplanning.com (React 18 + Vite + TypeScript, GitHub
  Pages)
- An `SEOHead` component already exists at `src/components/seo/SEOHead.tsx`
- Currently only `BlogListPage.tsx` and `BlogPostPage.tsx` use SEOHead
- The other **11 pages** rely on the generic `index.html` meta tags — Google
  sees the same title/description for every route
- The sitemap at `public/sitemap.xml` is auto-generated by
  `scripts/generate-sitemap.ts`

## Deliverable

### 2. Update Sitemap Priorities

Edit `scripts/generate-sitemap.ts` to use these priority tiers:

- **1.0:** Homepage `/`
- **0.8:** Planner `/planner` (renamed from `/` if applicable)
- **0.7:** Blog list `/guides`, individual blog posts `/guides/*`
- **0.6:** About `/about`, Help `/help`
- **0.5:** Support `/support`, Feedback `/feedback`
- **0.3:** Legal pages (privacy, terms, affiliate disclosure)
- **Exclude:** Settings, MapTest, NotFound (noindex pages should not appear in
  sitemap)

Run the sitemap generation script after updating to verify output.
